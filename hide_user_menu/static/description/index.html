Overview

  • Customization: Users can customize their Odoo experience by hiding specific menu items that they don't use or need, improving navigation efficiency.
  • Enhanced User Experience: By removing unnecessary menu items, users can focus on the features and functionalities that are relevant to their tasks, resulting in a more streamlined and intuitive user interface.
  • Flexibility: Administrators can easily manage and configure the visibility of menu items through the Odoo interface, without needing to modify the code directly.
  • Scalability: The solution can scale with the evolving needs of the organization, allowing for the addition or removal of menu items as the Odoo application expands or changes.

Install Module "Hide User Menu"

Configuration

Once Module Installed Go to the Settings menu, Look for 'Hide User Menu' in General settings. You can View User Menu Field



  • Now User Can select the Menus to hide and Click Save Button.
  • Once Page is reloaded, Selected Menus Will be Removed From Users Menu



  • Also If We Need to get back that Removed Menus , Just need to Remove in Users menu Field and click Save Button
  • Once page is reloaded Users can view Unselected Menus in Users Menu Items
